Astro-Swarm

2.5D Top-Down Shooter

Role: Engine and Gameplay Programmer

Engine: Custom Engine

Technologies: C++, Vulkan, EnTT, Gateware Library, Git

Repository: Astro-Swarm

Website: itch.io

Astro-Swarm is a 2.5D top-down arcade shooter featuring fully 3D models and environments with two-dimensional movement constraints. The game was built in a custom C++ engine from the ground up using Vulkan for rendering, EnTT for ECS architecture, and Gateware for input and windowing.

Key Contributions

• Developed player control logic, the lives system, and score tracking features with real-time updates.

• Designed and implemented the title screen and game flow system, enabling smooth transitions between gameplay states.

• Built and maintained the core engine structure, including rendering initialization, system architecture, and game loop integration.

• Identified and resolved gameplay bugs and logic issues, ensuring stable runtime performance.

• Used GitHub for version control, contributing to code integration, configuration management, and documentation efforts.
