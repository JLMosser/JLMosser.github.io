Relicboure

A roguelike monster-collector

Role: Gameplay and Systems Programmer

Engine: Unreal Engine 5

Technologies: C++, Blueprints, Perforce, Git

Website: itch.io

Relicborne is a turn-based, monster-collecting roguelike where players explore procedurally generated biomes, battle enemy teams, and capture powerful creatures using relic-based technology. Combat emphasizes party synergy, permanent monster death, and strategic resource use.

Key Contributions

• Built and maintained systems for combat flow, turn order, monster traits, passives, and relic capture logic.

• Wrote core documentation including glossary terms, flow charts, UI behavior, relic mechanics, and gameplay loop structure.

• Set up and managed source control using Perforce, creating custom ignore files and supporting a consistent workflow across Git and Unreal.

• Coordinated with design, programming, and art roles to ensure high-quality feature integration and efficient tool use.
